# kanduit-projects

Monorepo fuer Kanduit-Portfolio-Projekte mit klarer Trennung zwischen Quelle und Deployment.

## Struktur

- `portfolio/`  
  Projektquellen (Code, Datenpipelines, Streamlit-App-Code, Dokumentation).
- `docs/`  
  GitHub-Pages-Publish-Root fuer statische Projekte.
